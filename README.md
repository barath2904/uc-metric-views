# uc-metric-views

A Python CLI tool (`ucm`) that generates, validates, and deploys Databricks Unity Catalog metric view YAML definitions.

## YAML Directory

The `examples/` directory contains ready-to-use sample metric view YAML files. You can point `ucm validate` and `ucm deploy` at any directory containing your YAML definitions — `examples/` is just the default used by the GitHub Actions workflows.
